Timestamp Microservice

FCC Backend Project

Usage Examples:

    For Current Date: [project url]/api/

    For Specified Date With Format YYYY-MM-DD: [project url]/api/2015-12-25

    For Specified Date From Unix Timestamp: [project url]/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
